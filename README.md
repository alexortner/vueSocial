# vueSocial

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Firebase deployment

### Login to Firebase
```
firebase login
```
### Init Firebase 
```
firebase init
```

### Deploy to Firebase
```
firebase deploy
```
